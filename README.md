# Bioinformatics-LLM-Consulting-Platform

## 项目简介  
Bioinformatics-LLM-Consulting-Platform 是一款专为生物信息学领域打造的智能咨询平台，基于大语言模型（LLM）为研究者提供精准、个性化的数据分析解决方案。平台深度整合生物信息学研究场景，支持从需求提交到方案生成的全流程智能化，同时通过用户反馈与积分体系持续优化服务，助力研究者高效解决测序数据分析、基因注释、多组学整合等核心问题。


## 对生物信息学研究的核心价值  

1. **精准适配研究场景，降低技术决策成本**  
   针对生物信息学研究的多样性（如RNA-seq、单细胞转录组、多组学整合等），平台通过收集用户项目背景（`ConsultationContent`）和历史咨询记录，结合LLM的领域知识，生成贴合具体场景的分析方案。例如：  
   - 对RNA-seq数据用户，自动推荐从原始数据质控（FastQC）到差异基因筛选（DESeq2/edgeR）的完整流程；  
   - 对单细胞分析需求，优先匹配Seurat、Scanpy等工具的最佳实践。  
   无需研究者手动筛选海量工具文档，大幅缩短技术选型周期。

2. **实时交互与流式反馈，提升分析效率**  
   平台支持LLM流式回答（`RobotChatServiceImpl`），研究者可实时获取分析建议，避免等待全量结果的低效过程。同时，预设生物信息学高频问题（如“差异基因功能富集步骤”“单细胞数据降维方法对比”），通过前端`chat.js`的快捷提问功能，新手也能快速入门复杂分析。

3. **追踪研究热点，辅助前沿探索**  
   通过`HotWord`实体和`summaryMessage`方法，平台自动提取咨询中的高频关键词（如“空间转录组”“CRISPR筛选”），动态捕捉领域研究热点。研究者可通过热词趋势了解前沿方向，为课题设计提供参考。

4. **持续优化的闭环生态，适配领域发展**  
   用户可通过反馈功能（`FeedbackController`）提交方案改进建议，平台结合反馈数据迭代推荐模型。同时，积分体系（`points.js`）激励用户参与互动（如完善资料、提交反馈、使用智能对话），积累的积分可兑换生物信息学资源（如工具教程、数据集，对应`Gift`实体），形成“用户贡献-平台优化-价值反哺”的良性循环。


## 核心功能亮点  

1. **多模型智能对话引擎**  
   集成Qwen、Deepseek等主流LLM（`RobotChatController`），支持针对生物信息学细分场景的模型切换。例如，Deepseek在基因组注释任务中精度更高，Qwen在多组学整合分析中表现更优，用户可按需选择。

2. **个性化需求匹配系统**  
   通过`ConsultationContentController`允许用户更新研究背景（如数据类型、分析目标、预算范围），平台在生成回答时自动关联历史信息（`supplementConsultationInformation`方法），确保建议与项目上下文一致（如“基于您之前提到的小鼠肠道微生物数据，推荐使用QIIME2进行OTU聚类”）。

3. **反馈与积分激励机制**  
   - 用户提交反馈（每日1次，每小时1次限制）可获得积分，反馈内容直接用于优化LLM的领域适配性（`FeedbackServiceImpl`）；  
   - 积分可兑换生物信息学资源（如工具使用权、专家咨询时长），通过`Gift`实体管理兑换品，提升用户活跃度。

4. **研究过程追踪与回溯**  
   完整记录咨询历史（`queryByTimeRange`方法）和对话片段（`pullPieces`方法），支持按时间范围查询，方便研究者回溯分析思路、复现研究过程，符合生物信息学研究的可复现性要求。


## 技术架构  

### 核心技术栈  
- **后端**：  
  - 框架：Spring Boot（RESTful API设计、依赖注入）  
  - 数据层：MyBatis-Plus（实体映射与数据库交互，如`Feedback`、`HotWord`表管理）  
  - LLM集成：通过`RobotChatServiceImpl`实现流式对话与关键词提取，支持多模型切换  
  - 缓存与并发：Redis（对话片段存储、频率限制）、Redisson（分布式锁，避免重复提问）  

- **前端**：  
  - 交互层：JavaScript动态渲染（`chat.js`实现对话界面，`user.js`管理个人中心）  
  - UI组件：Bootstrap响应式设计，支持快捷提问、历史记录浏览等功能  

- **数据模型**：  
  - 核心实体：`Feedback`（用户反馈）、`HotWord`（研究热词）、`Gift`（积分兑换品）、`ConsultationContent`（咨询背景）  


## 使用流程  

1. **完善研究背景**：通过个人中心提交项目信息（数据类型、分析目标等），系统自动关联至后续咨询；  
2. **智能对话咨询**：在对话界面选择模型（Qwen/Deepseek），输入问题（或使用预设高频问题），实时获取流式回答；  
3. **反馈与优化**：对方案进行评价或建议，获取积分；  
4. **资源兑换**：使用积分兑换生物信息学工具或服务，加速研究进展。
